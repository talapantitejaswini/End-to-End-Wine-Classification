# 🍷 End-to-End Wine Quality Classification

## 📌 Overview

This project is an end-to-end Machine Learning application that predicts the quality of wine based on its physicochemical properties. The model is trained using the Wine Quality dataset and deployed as an interactive web application using Streamlit.

---

## 🚀 Features

* Predicts wine quality using a trained Random Forest Classifier.
* User-friendly Streamlit web interface.
* Accepts real-time user input for wine characteristics.
* Applies feature scaling before prediction.
* End-to-end machine learning workflow from model training to deployment.

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Pickle

---

## 📊 Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Train-Test Split
6. Feature Scaling
7. Model Training
8. Model Evaluation
9. Model Saving using Pickle
10. Streamlit Deployment

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/End-to-End-Wine-Classification.git
```

Navigate to the project folder:

```bash
cd End-to-End-Wine-Classification
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
